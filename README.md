# 📈 Bitcoin Signal Dashboard

A real-time Bitcoin buy/sell signal web app built with **Python**, **Streamlit**, and **Plotly**, using live price data from [CryptoCompare](https://min-api.cryptocompare.com/). The dashboard uses moving average crossover logic to help you identify potential trading signals.

👉 [Launch the App](https://btc-signal-dashboard-mjzbtxuq9dkpghrzrwrtru.streamlit.app)

---

## 🧠 Features

- ⏱️ Real-time Bitcoin price tracking  
- ⚡ Buy / Sell / Hold signals using moving average crossovers  
- 📊 Interactive Plotly chart  
- ⚙️ Adjustable short & long windows  
- 🧠 Streamlit UI that runs in the browser with no setup

---

## 🔧 Tech Stack

- **Frontend/UI**: Streamlit  
- **Charting**: Plotly  
- **Backend logic**: Python + Pandas  
- **Data source**: CryptoCompare API  
- **Hosting**: Streamlit Cloud

---

## 📦 Requirements

- Python 3.10+
- Packages (see `requirements.txt`):
  - streamlit
  - pandas
  - requests
  - plotly
